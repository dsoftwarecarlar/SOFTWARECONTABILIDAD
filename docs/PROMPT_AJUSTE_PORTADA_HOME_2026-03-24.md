Actua como auditor senior de UX para un sistema contable interno ya operativo.

Objetivo:
- revisar la portada principal del portal
- eliminar bloques redundantes o recargados
- dejar una entrada clara, limpia y facil de usar para personal contable

Reglas:
1. No mostrar arquitectura, runtimes ni lenguaje tecnico visible al usuario.
2. La portada debe conducir al usuario al area correcta, no duplicar pantallas de navegacion.
3. Si una seccion repite informacion que ya existe dentro del area, elimínala.
4. El encabezado visual debe verse compacto, sin vacios grandes ni tarjetas estiradas.
5. Prioriza:
   - logo claro
   - acceso principal al area
   - explicacion corta de uso
6. Evita frases promocionales o relleno.
7. Evalua especialmente:
   - composicion del bloque del logo
   - tarjeta de acceso
   - seccion "Acceso rapido"
   - jerarquia movil

Resultado esperado:
- portada sobria
- sin secciones duplicadas
- sin vacios visuales
- preparada para crecer con nuevas areas sin confundir al usuario
